## 迭代！迭代！迭代
### 使用迭代，而不是循环
在函数式编程，更推荐使用__迭代__：

```js
var results = _.map([1,2,3],function(elem){
  return elem*2;
}); // => [2,4,6]
```

而不是__循环__：

```js
var results = [];
var elems = [1,2,3];
for(var i=0,length=elems.length;i<length;i++) {
  results.push(elems[i]);
}
```
