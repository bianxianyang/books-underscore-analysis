### `_.wrap`：函数包装器，对指定函数进行包装，使函数的执行前后能融入更多业务逻辑
__源码__：
```js
_.wrap = function (func, wrapper) {
    // 借助于偏函数将`func`传递给wrapper
    return _.partial(wrapper, func);
};
```
